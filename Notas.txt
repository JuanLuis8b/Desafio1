Nota 1: "Una vez iniciada la adquisición, en cualquier momento se puede solicitar la 
información de la señal capturada. Esta petición se hará mediante la activación de 
un pulsador. Mientras se procesa la información, la adquisición de datos puede ser 
suspendida (alternativa: detenerse). Una vez se termine de procesar la información 
(y se entreguen las salidas,) la adquisición se debe reanudar."

Esta parte de la sección puede ser un poco ambigua, en especial porque no especifica
que se debe hacer con la adquisición de los datos una vez se presione el segundo pulsador 
; en cambio dice: "la adquisición de datos PUEDE ser suspendida", pero luego dice:
"Una vez se termine de procesar la información la adquisición se debe RENAUDAR."

Para esta situacion existen tres escenarios, el primero: se presiona el segundo pulsador,
se procesan los datos almacenados, se liberan y comienza inmediatamente con una nueva 
adquisición de datos hasta que se pulse de nuevo el segundo pulsador, el segundo: se presiona
el segundo pulsador, se procesan los datos almacenados, se  liberan y no comienza la adquisición
de datos hasta que se pulse de nuevo el primer pulsador y se repite el ciclo. 

El problema es entender a que se refiere que la adquisición PUEDA ser suspendida si luego dice que 
se DEBE renaudar, ¿cómo se suspende?
